Introduction to Flexbox

Flexbox is a powerful tool for achieving modern layouts in CSS. Below we will examine some of the different properties flexbox offers and how each affects flex content.

flex-direction

This property determines the order in which flex items are displayed relative to the order of the items in the markup, and whether they display inline or vertically stacked. This is also called the "main axis".

justify-content

This property determines where, relative to the flex container, the content is aligned along the main axis. Values of flex-start and flex-end depend on the direction set in flex-direction. space-between positions the first and last elements at the beginning and end of the container, respectively, and puts equal space between each of the elements. space-around gives equal space between elements, and the first and last elements have half the space before and after (respectively) as between elements. Note that items in contexts with flex-direction: column* are not affected by this property unless the flex container has a defined height.

flex-wrap

This property determines whether flex items can exist in multiple lines or whether they can only take up one line. Whether "line" means "row" or "column", and where the line originates and overflows depends on flex-direction and width or height settings of the container.

You can also use flex-flow as a shorthand property for flex-direction and flex-wrap like this: flex-flow: <flex-direction> || <flex-wrap>. How this affects in which direction lines of items stack, which is called the "cross axis"

align-items and align-self

align-items applies to the flex container and determines the default cross-axis alignment for flex items. align-self applies to individual flex items and determines that specific item's cross-axis alignment.

"baseline" aligns the items along the baseline of the text inside the items, putting the tallest item along the beginning of the cross axis and aligning everything else to it.

Horizontally positioning text content

Sometimes you will want to position text horizontally. You don't have fine-grained control over horizontal positioning of text through flexbox, but you can center it horizontally, or start the text from the left or right of the container. Use the container's justify-content property to change the horizontal origin of the text.

Vertically positioning text content

Sometimes you will want to position text vertically. You don't have fine-grained control over vertical positioning of text through flexbox, but you can center it vertically, or start the text from the bottom or top of the container. Use the container's align-items property to change the vertical origin of the text.
