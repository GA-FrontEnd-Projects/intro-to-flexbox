Introduction to Flexbox

Flexbox is a powerful tool for achieving modern layouts in CSS. Below we will examine some of the different properties flexbox offers and how each affects flex content.

flex-direction

This property determines the order in which flex items are displayed relative to the order of the items in the markup, and whether they display inline or vertically stacked.

justify-content

This property determines where, relative to the flex container, the content is aligned. flex-start and flex-end depend on the direction set in flex-direction. space-between positions the first and last elements at the beginning and end of the container, respectively, and puts equal space between each of the elements. space-around gives equal space between elements, and the first and last elements have half the space before and after (respectively) as between elements. Note that items in contexts with flex-direction: column* are not affected by this property unless the flex container has a defined height.
